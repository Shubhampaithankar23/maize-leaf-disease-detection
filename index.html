Maize Leaf Disease Detection

This project focuses on Explainable Deep Learning for detecting maize leaf diseases such as blight, rust, and gray leaf spot using image-based analysis.

Objectives

  • Automated maize disease detection
  • Explainable AI (XAI) for model transparency
  • Support for farmers and agronomists

Technologies

  • Python, Deep Learning
  • CNN / Vision Transformers
  • Explainable AI (Grad-CAM)
